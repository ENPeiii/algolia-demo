# Algolia 搜尋平台

## npm 套件安裝

- 使用 npm 安裝 Algolia 相關套件：
  - `algoliasearch`：官方 JS/TS 客戶端
  - `@algolia/client-search`：進階用戶端（可選）
  - Angular 專案可直接用 `algoliasearch`
- 安裝指令：
  - `npm install algoliasearch`

## Algolia 介紹

- 雲端搜尋平台，提供即時、快速的全文檢索服務
- 適用於網站、App、電商等多種場景
- 支援多語言、多平台

## 基本使用流程

- 註冊 Algolia 帳號，建立 Application
- 取得 Application ID 與 Admin API Key
- 建立索引（Index），上傳資料
- 前端整合搜尋功能

## 主要功能

- 快速全文檢索
- 支援 typo 容錯、分詞、排序、過濾
- 可自訂 ranking、faceting、highlight

## 進階主題

- 搜尋建議（Autocomplete）
- 多索引搜尋
- 安全性（API Key 權限控管）
- 分析與監控
